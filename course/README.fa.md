هر مرحله، فایل راهنما، solution و کد `AppHost.cs` خودش را دارد. `src/` کد مشترک قراردادها، سرویس‌ها و مکانیزم‌های مشاهده‌پذیری را نگه می‌دارد؛ بنابراین هر solution قابل اجراست، بدون آن‌که سه سرویس را برای هر درس کپی کنیم.
